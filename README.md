## Neural Network Exercises

A collection of deep learning projects demonstrating the implementation of **Artificial Neural Networks (ANNs)** and **Convolutional Neural Networks (CNNs)** for classification tasks. These exercises cover structured data as well as image classification using popular benchmark datasets.

# Projects

1. Sonar Dataset Classification (ANN)
Model : Artificial Neural Network (ANN)
Dataset : Sonar Dataset
Task : Binary classification to distinguish sonar signals reflected from rocks and mines.
Concepts : Data preprocessing, binary classification, ANN architecture, model evaluation.

2. Digits Dataset Classification (ANN)
Model : Artificial Neural Network (ANN)
Dataset : Digits Dataset
Task : Multi-class classification of handwritten digits (0–9).
Concepts : Image flattening, dense neural networks, multi-class classification.

3. Fashion Dataset Classification (ANN)
Model : Artificial Neural Network (ANN)
Dataset : Fashion-MNIST
Task : Classify clothing items into 10 categories.
Concepts : Image preprocessing, dense neural networks, classification.

4. Handwritten Digits Recognition (ANN)
Model : Artificial Neural Network (ANN)
Dataset : MNIST
Task : Recognize handwritten digits using a fully connected neural network.
Concepts : Image classification, dense layers, activation functions.

5. Handwritten Digits Recognition (CNN)
Model : Convolutional Neural Network (CNN)
Dataset : MNIST
Task : Recognize handwritten digits using convolutional layers for improved accuracy.
Concepts : Convolution, pooling, feature extraction, deep learning.

6. Flower Classification (CNN)
Model : Convolutional Neural Network (CNN)
Dataset : TensorFlow Flower Photos Dataset
Task : Multi-class image classification to identify flower species (Daisy, Dandelion, Roses, Sunflowers, and Tulips).
Concepts : Image preprocessing, data augmentation, CNN architecture, multi-class classification, model evaluation.
